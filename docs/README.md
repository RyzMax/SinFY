- SinFY это сайт для воспроизведения, а также скачивания музыки с модерацией.
<h2></h2>
 <h3>🛠️ Его функции:</h3>

- Возможность **создавание аккаунта для его использования, комментирования, а также выкладывания музыки**
- Возможность **модерировать музыку, а также комментарии**
- Возможность **менять свою аватарку, описание, ник**
- Возможность **комментирование** и в следствие чего -> **жалоба на комментарии**
- Возможность **фильтрации** музыки
- Возможность **подписатся на создателя**
